# AtifshaCode Website

A professional services website for a programming/software development business that builds custom digital systems for shops, cafés, and other businesses.

## Stack

- **Runtime**: Node.js 20
- **Server**: Express.js (serves static files)
- **Frontend**: HTML, CSS, JavaScript (no framework)
- **Port**: 5000

## How to Run

The app is configured to start automatically via the **"Start application"** workflow.

To run manually:
```bash
node server.js
```

## Project Structure

```
├── server.js          # Express static file server
├── public/
│   ├── index.html     # Main website page
│   ├── style.css      # All styles
│   └── app.js         # Interactivity (nav, form, animations)
└── package.json
```

## Pages / Sections

- **Hero** — tagline, CTA buttons, stats
- **Services** — 6 service types (Shop POS, Café App, Web App, Mobile App, Business Tools, E-Commerce)
- **Technologies** — tech stack pills
- **How It Works** — 4-step process
- **Portfolio** — 3 sample projects
- **Contact** — contact form + details

## User Preferences

- Business-focused content for shops, cafés, and general businesses
- Dark theme, modern design
